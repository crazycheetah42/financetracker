 # Finance Tracker

A simple desktop finance tracker built with Python and Tkinter. Add income and expenses, view your current balance, and review your transaction history.

## Features

- Record income and expenses with descriptions
- Store transactions in a local file
- View the calculated balance and transaction history
- Use a dark-themed Tkinter interface

## Usage

Run the application from the project directory:

```bash
python main.py
```

Enter a description and amount, select **Income** when applicable, and click **Submit**. Transaction data is saved locally.
